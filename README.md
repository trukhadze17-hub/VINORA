
# VINORA — AI Automotive Platform

Full premium starter architecture for an AI-powered auto import platform.

## Features
- AI VIN Analysis
- AI Damage Detection
- Import Cost Calculator
- Admin Dashboard
- Auction Monitoring
- User Authentication
- AI Chat Assistant

## Tech Stack
- Next.js 15
- TailwindCSS
- TypeScript
- PostgreSQL
- Prisma ORM
- OpenAI API

## Start
npm install
npm run dev
